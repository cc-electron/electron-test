# chat-os

### 运行环境

```
Node: v14.17.0
Vue-cli: v4.5.0

```

### create app
```
vue create app
cd app
vue add electron-builder
```

## Project setup

```
npm install
node > v12
```

### Compiles and hot-reloads for development
```
npm run serve
-- electron-vue
npm run eserve
```

### Compiles and minifies for production
```
npm run build
-- electron-vue
npm run ebuild
```
